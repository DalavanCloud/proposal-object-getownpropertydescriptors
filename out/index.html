Object.getOwnPropertyDescriptors

1Object.getOwnPropertyDescriptors( genericObject )#

When the getOwnPropertyDescriptors function is called, the following steps are taken:

  1. Let obj be ?ToObject(O).
  2. Let keys be the result of calling the ?[[OwnPropertyKeys]] internal method of obj.
  3. Let descriptors be the result of the abstract operation ?ObjectCreate with the intrinsic object %ObjectPrototype% as its argument.
  4. Let gotAllNames be __false__.
  5. Repeat while gotAllNames is __false__,
    1. Let next be the result of ?IteratorStep(keys).
    2. If next is __false__, then let gotAllNames be __true__.
    3. Else,
      1. Let nextKey be ?IteratorValue(next).
      2. Let nextKey be ?ToPropertyKey(nextKey).
      3. Let desc be the result of calling the ?[[GetOwnProperty]] internal method of obj with argument nextKey.
      4. Let descriptor be ?FromPropertyDescriptor(desc).
      5. Let status be the result of ?CreateDataProperty(descriptors, nextKey, descriptor).
  6. Return descriptors.
