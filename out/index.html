Object.getOwnPropertyDescriptors

1Object.getOwnPropertyDescriptors ( O )#

When the getOwnPropertyDescriptors function is called, the following steps are taken:

  1. Let obj be ? ToObject(O).
  2. Let ownKeys be the result of calling ? obj.[[OwnPropertyKeys]]().
  3. Let descriptors be ! ObjectCreate( %ObjectPrototype%).
  4. Assert: descriptors is an extensible ordinary object with no own properties.
  5. Repeat, for each element key of ownKeys in List order,
    1. Let desc be the result of ? obj.[[GetOwnProperty]](key).
    2. Let descriptor be ! FromPropertyDescriptor(desc).
    3. Let status be the result of ! CreateDataProperty(descriptors, key, descriptor).
  6. Return descriptors.
